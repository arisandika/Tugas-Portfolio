Ari Sandika's Blog - Personal Blog

Fullstack Development

Ari Sandika's Blog - Personal Blog is a dynamic personal blog platform built using Laravel for the backend, offering a robust RESTful API with JWT authentication for secure user management. The frontend is powered by Next.js, delivering a fast and responsive user experience. The platform enables users to create, edit, and manage blog posts with ease, providing features like role-based access control and a clean, intuitive interface. This project blends the power of modern web development technologies to create an efficient and scalable blogging experience.
